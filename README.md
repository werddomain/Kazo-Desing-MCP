# Kazo-Desing-MCP
Kazo Design MCP est une extension VS Code pour la création de diagrammes vectoriels assistée par IA (MCP). Dotée d'un moteur de dessin performant en C# (.NET 10 Blazor WASM), elle permet à Copilot de générer des schémas SVG, éditables par l'utilisateur et sauvegardés automatiquement en documentation technique riche (Markdown + JSON).
