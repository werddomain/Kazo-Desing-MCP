Show AI What You Want to Build

Create quick sketches and mockups to visually express your design intentions to AI assistants like GitHub Copilot. Powered by a high-performance C# (.NET 10 Blazor WASM) drawing engine, with automatic export to SVG and rich Markdown documentation.

Key Features

Hybrid Architecture

The power of .NET 10 Blazor for drawing, the fluidity of TypeScript for VS Code integration.

Native SVG Format

Designs are pure SVG files. No proprietary format, easily editable and versionable.

Automatic Documentation

Each design generates a companion Markdown file including the AI prompt and JSON metadata.

MCP & Copilot Integration

Use natural language to ask AI to generate or modify your system architectures.
