Kazo Design MCP

Visual Communication for AI Assistants

Create quick sketches, mockups, or wireframes to visually express what you want AI to build. Powered by a high-performance C# rendering engine (.NET 10 Blazor WebAssembly).

VS Code Extension .NET 10 Blazor WebAssembly

Features

üé® Quick Sketch Editor

Create rapid mockups and wireframes directly within VS Code. The editor provides a simple, intuitive interface for expressing your design ideas to AI assistants.

üß† AI Communication Tool (MCP)

The Model Context Protocol (MCP) integration enables seamless communication between you and your AI assistant. Your sketches become visual context that helps the AI understand exactly what you want.

üîß Shape Tools

  • Rectangle: Create boxes for layouts, containers, or buttons
  • Circle: Add circular elements for icons, avatars, or decorations
  • Line: Draw connections, separators, or pointing elements
  • Text: Add labels to explain different parts of your sketch

üíæ Smart Export Format

When saving a design, two files are created automatically:

  • SVG file: Vector image for documentation, websites, or further editing
  • Markdown file: Companion documentation with title, preview, prompt, and JSON data

üéØ Interactive Canvas

  • Drag & Drop: Move elements freely on the canvas
  • Selection: Click to select elements for editing or deletion
  • Grid: Visual grid overlay for precise alignment
  • Zoom: Ctrl+scroll to zoom in/out
  • Pan: Scroll to pan around the canvas

How It Works

Ask the AI to create something (e.g., a web page, a component, a UI layout)

2

The AI invites you to create a visual sketch of what you have in mind

3

4

Click the "Confirm & Return to AI" button when your sketch is ready

5

Generate

Getting Started

üì¶ Installation

From VS Code Marketplace

  1. Open VS Code
  2. Go to Extensions (Ctrl+Shift+X)
  3. Search for "Kazo Design MCP"
  4. Click Install

From VSIX File

  1. Download the .vsix file from Releases
  2. In VS Code, go to Extensions
  3. Click ... menu ‚Üí "Install from VSIX..."
  4. Select the downloaded file

üöÄ Usage

Opening the Editor

  1. Open Command Palette (Ctrl+Shift+P)
  2. Type "Kazo Design" and select:
    • Kazo Design: Open Editor
    • Kazo Design: New Design

Creating Shapes

  1. Click a shape tool in the toolbar
  2. The shape appears on the canvas
  3. Drag shapes to reposition
  4. Click to select, Delete to remove

Available Commands

Command Description
Kazo Design: Open Editor Opens the diagram editor panel
Kazo Design: New Design Creates a new blank design
